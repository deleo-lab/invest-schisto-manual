Downloading InVEST

The Schistosomiasis Tool is built on top of NatCap’s InVEST Workbench software. Before using the tool, you will have to download InVEST Workbench.

We’ve created a custom version InVEST to support the custom functionality needed for the schistosomiasis model. When users download InVEST to run the schistosomiasis plugin, they will be using this custom version of InVEST. In the future this might not be necessary and users can use regular InVEST for the schisto plugin, but the following custom functionality makes it necessary at the moment.

Windows

  1. Windows users must confirm that the Microsoft Visual C++ Redistributable is installed. To check if you have the compiler installed you can:
    1. Open Add or Remove Programs in Settings \> Apps \> Installed apps
    2. Search for C++
    3. You’ll see something like Microsoft Visual 2015-2019 Redistributable...
    4. If you don’t see anything like that you’ll need to install the compiler
  2. To install the compiler
    1. Go to this link
    2. Download and install Visual Studio 2026 using the Community option.
    3. Alternatively you can just install Download and install the Microsoft Visual C++ Redistributable from this link. Both the x86 and x64 versions.
    4. Open the .exe files and follow to installation prompts.
  3. Download and install the InVEST workbench
    1. Download custom version of InVEST file with win32 in the name. 
    2. Open your downloads folder and open the .exe file
    3. A box Windows protexted your PC will open. Click More info and Run anyways
    4. Follow the installation prompts

Mac

  1. Download and install the InVEST workbench
    1. Download custom version of InVEST file with darwin in the name.
    2. Open your downloads folder and open the .exe file
    3. Once the file has downloaded, open it and click Agree when the license prompt appears.
    4. A window will appear telling you to drag InVEST to Applications. Drag the InVEST app to your Applications folder.
    5. IMPORTANT: Launch InVEST for the First Time. MacOS security will block the app on first launch. To allow it:
    6. Open Applications
    1. Right-click the InVEST app \(\to\) choose Open
    2. In the popup, click Done (the app will not open yet)
    3. Open Applications \(\to\) System Settings \(\to\) Privacy & Security
    4. Scroll to the Security section and you should see: InVEST Workbench was blocked to protect your Mac. Click Open Anyway.
    5. Click Open Anyway again on the pop up box.
    6. Enter your Mac username + password when prompted.
  1. The InVEST Workbench will now start as usual. This approval is only needed the first time.

Installing the Schistosomiasis Plug-in

Once InVEST Workbench is downloaded, you will need to open it and add the Schistosomiasis Plug-in to use the Schistosomiasis Tool.

  1. Open InVEST Workbench. In InVEST Workbench click the three line menu icon \(\to\) Manage Plugins \(\to\) Add Plugin.

  2. Copy and paste the following for git URL: https://github.com/natcap/invest-schistosomiasis.git. Check the box to acknowledge the risks associated with installing the plugin. Leave everything else blank. Click Add. This step will take several minutes. It is setting up an isolated python environment, downloading, and installing the schistosomiasis model. You only need to do this step once.

  3. The Schistosomiasis plugin is now available in the workbench.

  4. The elements of the workbench are described in the InVEST documentation.
